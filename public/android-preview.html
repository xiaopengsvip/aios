好的！这是一个 Kotlin 实现的快速排序： ```kotlin fun quickSort(arr: List): List { if (arr.size <= 1) return arr val pivot = arr[arr.size / 2] val left = arr.filter { it < pivot } val mid = arr.filter { it == pivot } val right = arr.filter { it > pivot } return quickSort(left) + mid + quickSort(right) } ``` 时间复杂度：平均 O(n log n)，最坏 O(n²)
